# 地球图鉴收集 - Earth Atlas Collection

一个美观的图鉴收集Web应用，支持拍照/上传图片，通过AI生成精美的图鉴卡片。

## 功能特性

### ✨ 核心功能
- 📸 **拍照/上传**: 支持移动端拍照和本地图片上传
- 🎨 **AI生成图鉴**: 模拟AI处理流程，生成美观的图鉴卡片
- 🖼️ **图片墙展示**: 瀑布流布局，支持多种卡片尺寸（1x1, 2x1, 2x2）
- 🔍 **图片预览**: 全屏图片查看器，支持缩略图导航

### 📱 跨平台支持
- **PC端**: 完整的桌面体验，键盘快捷键支持
- **移动端**: 响应式设计，触摸手势支持
- **平板**: 自适应布局

### 🎯 用户体验
- **懒加载**: 图片和内容的渐进式加载
- **无限滚动**: 自动加载更多内容
- **加载动画**: 精美的AI处理动画效果
- **手势支持**: 移动端左右滑动切换图片

## 技术栈

- **前端**: 纯 HTML/CSS/JavaScript
- **布局**: CSS Grid + Flexbox
- **动画**: CSS Animations + Transitions
- **响应式**: CSS Media Queries

## 快速开始

1. 直接在浏览器中打开 `index.html`
2. 点击"拍照/上传图片"按钮
3. 选择或拍摄图片
4. 等待3秒AI处理动画
5. 查看生成的图鉴

## 功能说明

### 首页
- 用户头像显示在左上角
- 中央的拍照/上传按钮
- 下方展示图片墙列表

### 图片上传
- PC端: 点击按钮选择本地图片
- 移动端: 可选择拍照或从相册选择
- 上传后显示3秒加载动画

### 图片查看器
- 全屏展示大图
- 底部缩略图导航
- 分享和下载功能
- ESC键或点击背景关闭

## 交互操作

### PC端
- `ESC`: 关闭图片查看器
- `←/→`: 切换上一张/下一张图片
- 鼠标滚轮: 浏览图片墙

### 移动端
- 左右滑动: 切换图片
- 双击: 缩放图片
- 下拉刷新: 重新加载（需实现）

## 开发说明

主要颜色变量定义在 `styles.css` 的 `:root` 中：
```css
--primary-color: #4A90E2;
--secondary-color: #7B68EE;
--background-color: #f5f7fa;
```

### 添加新功能
在 `app.js` 的 `EarthAtlasApp` 类中添加新方法

### 响应式断点
- 移动端: < 480px
- 平板: 480px - 768px
- 桌面: > 768px
